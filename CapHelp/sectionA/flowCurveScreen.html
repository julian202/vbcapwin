Flow Curve Screen

The screen displays the wet and dry flow curves and a curve showing the half dry flow rate. The intersection of the half dry flow rate curve and the wet curve is the mean flow pressure. In multi-graph mode, the Flow Curve Screen and Text Screen are shown for each of the selected files.

Continue
Exits the current graph and moves on to the text report is shown next.

Cancel
Returns the user to Report Setup.

The Text Report displays textual report data for a selected data file. In multi-graph mode, the first Text Screen may contain a list of the selected files.
