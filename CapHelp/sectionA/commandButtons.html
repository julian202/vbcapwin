Command Buttons

Begin(Single)
Click BEGIN after all report and graph options have been selected to create the data analysis and graphing reports. See the section on data report screens.


Begin(Multi)
The Begin (Multi) option allows the user to start reports using previously saved files in the File List. In parentheses shown on the BEGIN (Multi) option, provides the number of data files selected to be displayed. This number can be changed by clicking on the File List button and adding or deleting data files from the list shown.


File List
Contains a list of data files that is maintained until changed by the user. This option is helpful for preserving a list of files needed for repeated reports or calculations.

Right Click to clear list.

Add Files to list
Takes user to a file selector and allows the user to pick data files for various test. Up to 5 files can be selected

Remove Selected Files
Allows user to delete files from the list.

Done
Closes the window and brings you back to the main screen

Settings
Takes you to Caprep Preferences Main Screen and allows the user to customize the text reports and graphs produced. This will be explained in more detail in the report graphing screen section.

Quit
Closes the Caprep program and returns to the last active window.

MULTI-Graphic Option (up to 5 data sets simultaneously)
Allows up to 5 data files of the same type to be printed on a single graph. The Color Lines option is automatically selected. Graphs displayed with the Multi-Graphic Option are automatically shown in color on the screen and on color printers.
