Special Options

Contains Pore Numeric reports and graphs, which are usually not used. This tab is normally disabled. Settings

General

EMF Size Factor
This is a printing option for graphics saved as an EMF file. The factor determines how much of the opened window gets saved. (For example, 0.75 would correspond to 75% of the window. Usually the factor will just be set as 1 meaning that 100% of the window will be saved as a graphic file.) Older versions of Windows may need different factors.

Special Permeability Type
Most of the time this option is left at zero. If the user wishes to calculate permeability in an alternate way, he can change this option to 1. For more information on this option, please contact the PMI offices.

Fonts
Allows you to pick the font used in the graphs and reports. The left-hand column defines the font properties of the Caprep software itself and text printed on graphs. The other column is used for text in text reports. The main font defaults to Arial. The font used for text reports should be a monospaced font in order to preserve the correct report formatting; the default font is Courier New.

Text


Allows you to pick what information is shown in the Summary Sheet and specifies the units used to report flow.

Alternative Distribution Calculation
The pore size distribution is an estimate of the relative abundance of pores taken at each point in test. This information is acquired by looking at how much the air flow changes as the diameter of the pores change. The pattern of air flow change will depend on how the number of pores of a certain diameter are spread out across the sample. This option just allows the user to opt to calculate this distribution in an alternate way. Normally it is not selected. For more information contact PMI.

Graphs

Scaling
These options determine how graphs are scaled. Scaling determines the ranges of the axis for each graph.

Last Scale
Uses the scaling information stored during the last Manual Scaling or Auto Scaling report. This method may miss data points if the data does not fit the same range as the last data set.

Manual Scale
Adjusts the settings for each graph manually.

Auto Scale (Recommended)
Automatically adjusts the scaling to graph all of the data.

Fitting

Histogram
Displays the data in a histogram.

Fitted x-y plot
Displays the data in an x-y plot; creates a line graph instead of a bar graph; the points displayed will correspond with the histogram boxes if a bar graph were made instead.

Max number of fitting intervals
Sets the max number of histogram bins if using a histogram display and the max number of intervals for between points if using an x-y plot.

X Axis for Pore Distribution
Categorizes pore sizes by the amount of pressure it takes for each pore diameter to allow flow to pass.

Pressure
Categorizes pore sizes by the amount of pressure it takes for each pore diameter to allow flow to pass.

Diameter
Categorizes pore sizes by their diameter.

Distribution as f (D^2)
Changes the way that Pore Size Distribution is calculated. By squaring the diameter value, the distribution curve is more pronounced. The peaks appear to be much greater that if viewed with this option turned off.

Y Axis for Permeability

Cc/s
(cubic centimeters per second)

L/m/cm^2
(liters per minute per unit area of the sample vs pressure)

Show Data Points
A data point marker representing each data point is shown on the graph.

Color Lines
Displays graph lines in color.

Normalize Data
Scales the data points down so that they fit onto a normal plot. Sets the maximum point at 100 and adjusts the other points accordingly. Extra Liquid Perm. Printing

Graphics Based vs Font Based Output
Determines how the printer or plotter should handle output data. The font and word-based options should work on any relatively modern computer while the graphics and character options were made for much older versions.

Page Margins
Sets the page margins for printed reports. A twip is a printing term for width and is equal to 1/1440 of an inch. The default values for this option are 200 for the right and left margins, 240 for the top and 80 for the bottom.

Set up Reports for Three Ringed Binder
Moves the text to the right to make space for binder holes.

Line Width on Graphs
Allows the user to set the thickness of the lines on the graph; can be made more or less visible by inputting higher or lower numbers.

Print Test Page
Prints the test page using the specific options set above.

Data Report Screens
