Report Graphing

The menu bar has several options for viewing and outputting the data:

Back/Next
Go to the previous or next report in the series.

Done
Quit viewing the current graph and return to Report Setup.

Print
Prints the current graph.

Zoom In
Zooms in on a section of the graph. The area selected in normal view is scaled to fit the graphing window. Place the mouse in one corner of the area to be scaled. Click and hold the left button. Move the mouse to the opposite corner of area to be scaled and release.

Manual Zoom
Enter new values for the minimum and maximum ranges of each axis. Click the OK button to view the new plot.

Zoom Out
Redraws the graph displaying all data.

Points On/Off
Displays a marker on the graph for each data point in the file. Select the option again to turn it off.

Color Lines On/Off
Displays each graph line in a different color and makes the graph more readable. Select the option again to turn it off.

Grid Lines On/Off
Displays grid lines. Select the option again to turn it off.

Abort
Exits the report
