Data Averager

The Data Averager combines up to five data files into a single file with averaged values.

Use the Add File button to select the files to be averaged. All the files must be of the same type of test (Porometry, Permeability, etc.).

Click Begin to start averaging. The data in the selected files is combined into a single, temporary file. This file is sorted, placing similar data points together.

An average curve is then constructed for the data points in this file and Curve Fit is run. Refer the section Curve fitting.

After the files have been averaged a file selector is displayed. Enter a file name to store the averaged data.

Begin Starts the Averager using the data files selected.
Help Start the CAPWIN Help System.
Close Exit the Data Averager without combining the selected files.
Add File Choose up to five data files to average.
