Pressure Step List

This option allows the user to select a series of pressures that data will be recorded from as the test is being run. (When tests are run, the pressure is incremented in steps; this option allows you to pick out various pressure values at which the data will be recorded.) The test will record only at the pressure points specified in the list.

The directions on the right half of the screen can be followed to complete this option. To enter pressures, type in the values in the entry field above the ADD button. Use the DELETE button to remove any pressure values. Use the CLEAR LIST option to clear the list of entered values.
