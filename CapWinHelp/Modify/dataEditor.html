Data Editor

The Data Editor option allows information in the data files to be edited.

Menu Bar


FILE Load Loads a data file or a portion of one into the editor. The user will be taken to a file selector:

File

  • Load    Loads a data file or a portion of one into the editor. The user will be taken to a file selector:
  • Save    Saves the data to a file. User will be taken to a file selector box
  • Close    Closes the editor and returns to the CAPWIN Main Screen.
  • Import
    • Dry Curve  Loads only the dry curve from a data file. If this option is used while another data file is open in the editor, the imported dry curve will replace the dry curve in the current file.
    • Wet Curve  Load only the wet curve from a data file. If this option is used while another data file is open in the editor, the imported wet curve will replace the wet curve in the current file.
  • Export  Exports the dry curve from a porometry data file. The dry curve is saved as a gas permeability file.

REFRESH
Updates the graphical display of the dry and wet curves.

HELP
Starts the CAPWIN Help System.

The display box at the top of the screen shows a graphical representation of the data currently stored in the editor and modifications made. When the mouse is moved over the graph, the pressure and flow rate of the data at the particular cursor point on the curve are displayed below (P indicates pressure and F indicates flow).

The list below the graph display box is comprised of the command prompts that can be activated when the cursor is within the graph box. To use these commands, place the cursor at the position to be changed, then type the appropriate letter you wish to see:

(B)ubble Pt.   Only usable with wet curve data. This command allows the bubble point to be relocated. All wet phase data to the left of this point is lost if the bubble point is moved to a point later in the time span of the graph.

(T)rim    Removes the data points to the right of the cursor's current position.

(Z)oom   Zooms in on the graph. The cursor position marks the upper right corner of the area to magnify. The data not shown on the rescaled graph is NOT lost.

(R)estore   Restores the data to its original state as it was loaded.

(M)ove Dry   Allows the dry curve to be rescaled vertically while keeping the intercept at "0,0".

The bottom half of the screen contains modifiable information about the test setup. This is information that was entered on the automated test setup screen:

Fluid Type of wetting fluid used in test.
Date Date test is being run.
Lines 1 & 2 User information about the sample.
Sample ID Information that identifies the sample.
Permeability Gas/Liquid The gas or liquid used to run the permeability tests.
Surface Tension The surface tension of the liquid.
Diameter of Sample The diameter of the tested sample.
Thickness of sample The thickness of the tested sample.

At the bottom of the screen there is a slider that displays different parameters of the test, beginning with Operator. If you click on the right hand arrow, more parameters will be displayed (.Lot Number, Type of Test, Wet Parameter, and finally Dry Parameter) The entry box above the slider allows you to change the parameters or their labels. These parameters are the same that have been entered in the Autotest Screen. For more information on these parameters, please refer to the AutoTest Section .

Below the movable arrow there is an option for Tortuosity Factor. The default value for this factor is 0.715. If you would like to change this value click on the check box. Then enter a new value in the designated space. For more information on the Tortuosity Factor please refer to the section of the Advanced Settings page.
