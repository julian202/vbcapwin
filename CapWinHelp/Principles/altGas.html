Using Alternate Input Gases

Beginning with Version 6.70.48, Capwin supports easy switching among different input gases. Flow conversion factors are used for each gas to more accurately take the properties of the source gas into account.

The default gas is assumed to be air/nitrogen, which is the only gas predefined for the system. New gases must be added manually to a definition file in the main Capwin folder (GASFLOWCONVERSION.INI).

The definition file may be opened in any basic text editor, such as Notepad. The default file will read as follows:

[main]
gas1=Air/Nitrogen
conversion1=1
current=1

Each gas is defined by two lines with the keys "gas1" and "conversion1", "gas2" and "conversion2", etc.

Adding a new gas definition

To add a new gas definition to Capwin, follow these steps. We will use an imaginary gas, nullane, as an example:

  1. Open GASFLOWCONVERSION.INI in a text editor.
    Each defined gas is numbered. Two lines are used for each gas, of the form "gas"n and "conversion"n, where n is the number of the new gas.

  2. Add a line defining the name of your new gas, with the next available index:
    gas2=Nullane

  3. Add a line defining the gas's flow conversion factor in terms of nitrogen. Use the appropriate index:
    conversion2=0.54148

  4. Close and save the text file.
    If Capwin is running, quit it and restart the program. You will see that an extra gas indicator box has appeared in the main window:
    The gas indicated will be used for any tests run on the porometer.


Selecting a defined gas for a test

You may switch among defined gases by clicking on the name of the gas in the indicator box. Click on the name of the gas in the Gas box. A selector box will appear with a list of defined gases:

Select the name of the gas you wish to use and click OK.

The indicator box will now show the name of the new gas. This gas will be used in any tests run on the machine.
