Adjust Flow Parameters

The procedure for calibrating flow parameters has two steps. The first step builds a table of low flow controller settings as flow rate is seen. This information is stored in a file (CAPCAL.D8A). This table is used when performing a bubble point test to determine what controller setting will produce the desired bubble flow.

The second step is to slowly increment the opening of Valve 2 until the first flow rate is detected. The valve position is read and the count value is stored in a file (CAPCAL.D8A). This value is used to keep the system from opening the valve too far and releasing too much pressure at once. This calibration must be run prior to running the Lohm calibration.

Calculate Lohm and Max Air Flow

The Lohm factor is a correction value for the difference in air pressure between the pressure transducers and the sample chamber. It must be calibrated before any tests are run on the instrument. A separate lohm calibration should be run for each sample chamber or set of adapter plates being used.

The maximum air flow calibration uses the current pressure in the instrument to find the maximum amount of air flow that can be obtained through the instrument with out a sample in the chamber. When complete, both values (Lohm and Max air flow) are saved to a file (CAPWIN.INI).

Max-Liquid Flow

The maximum liquid flow calibration finds the maximum amount of liquid flow that occurs during the Liquid Permeability test. When complete, the value is saved to a file (Capwin.ini).

Calibrate Regulator

During this calibration, the regulator is set to certain count values and the correlated pressures are found for the count values selected. (The regulator's function is to control pressure going into the system and convert that pressure to counts.) A table is then made out of this data. This particular calibration should be run if the inlet pressure of the machine changes (for example if the machine has been moved to a new location, with a new air source.)

Chamber Volume

Chamber volume is calibrated for the Pressure Hold Test and Micro flow Test. During this calibration, the total volume of the machine's interior (including all tubing and connections) is calculated. In the Micro flow Test itself, the flow rate is calculated from volume and pressure; thus this calibration makes sure the volume part of the calculation is accurate.

Load Backup Lohm Table

The Lohm table created in the Calculate Lohm and Max Air Flow option is stored in the "Parms" directory. If the user wishes to use another table of values besides the one already made, this option will allow you to select another pre-made table. A specific table should be used for every different size adapter plate and chamber used.
