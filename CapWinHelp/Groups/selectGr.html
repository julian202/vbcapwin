Select Group

Use this function to switch among existing groups.

A group contains the measurement units and preferences for the auto test setup used for the last automated test run while the group was active. Each time an automated test is run, any changes made to the auto test setup are automatically saved.

Click Group → Select Group from the CapWin Main Screen menu bar. The box shown above is displayed. Select the desired group and click OK.

Returns to CAPWIN Main Screen without changing the current group.

Makes the selected group the active group and returns to CAPWIN Main Screen.
