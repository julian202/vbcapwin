Example: Using Groups

The following is an example that will familiarize you with the usage of groups. You will be creating a group for lot number 42, sample 1. This example will be deleted when you are through.

1.From the Main Screen, select Group→ New Group.

2.Type lot42 in the box next to New Group. Click OK.

You are returned to the Main Screen. "lot42"

Shows in the Group Name box.

3. Select Autotest from the Execute Menu.

The Autotest screen appears. (The buttons at the top of this screen are explained in the Autotest section of the Execute Menu heading, but their use is not necessary for the task at hand.)

4. Click on the blank space next to Output File Name. The Output Data File file window appears. This is how the window will be if CAPWIN was installed in the default location. If you opted for a different path, locate the Data folder under the Capwin folder and select it.

5. Click the New Folder icon at the top of the screen. Type lot42 in the input box. Hit save.

A lot42 folder appears below the data folder. In the input box, type in sample1, overwriting lot42. Click ok. The sample1.cft output data file is created. The path and file names are displayed in the Output File Name box.

6. Click on the blank space next to End User. A dialog box appears. Type Customer #1 in the input box.

Click OK.

7. The Autotest options screen displays Customer #1 as the End User.

8. Following the same procedure as Step 8, enter the information shown in the illustration at right for the next 4 lines.

9. Leave Fluid as Porewick and Surface Tension as 16 dynes/cm.

When necessary, Fluid and Surface Tension can be changed in the same manner as the previous items. The values shown on the Autotest options screen are printed on the report. Changing the parameters for the test is explained in the test procedures.

10. Double-click on Dry Curve Testing Parameters File.

11. The DRY Parameter File window appears. The file selector appears as shown if CAPWIN was installed in the default location. If you opted for a different path, locate the Parms folder within the Capwin folder and select it.

Click on bp-deflt.tpf.

Click OK.

Both the Dry Curve and the Wet Curve Testing Parameters File options change to bp-deflt.tpf. This is because usually both the dry and wet phase of the test use the same parameters.

12. Double-click on Wet Parameter.

A dialog window appears. It will ask you if you would like to select a WET Parameter file that is different from the DRY.

Select Yes.

13. The WET Parameter File selector appears.
The file selector appears as shown if CAPWIN was installed with the default path. If you opted for a different path, locate the Parms folder under the Capwin folder and select it.Click on Bp-default.tpf.
Click Open.

The Wet Curve Testing Parameters File is now Bp-default.tpf.

14. Next, click on the space next to the Type of Test option. The Test Selection screen appears.
Select Bubble Point in the yellow list and Automatic Detection in the green list.
Click OK.

The items not used by the selected test will not appear on the Autotest options screen.

15. Click on the space next to Type of Test again.
Select Capillary Flow Porometry from the yellow list and Wet up/Dry down from the green list.
Click OK.

16. Diameter, Cylinder Length and Thickness, Absolute Density, and Mass are discussed in the section of the manual dealing with testing procedures, (link here). Do not make any changes to them for now.

17. If this were an actual test, you would click Start Test to save this setup and start the test. The next time this group was selected, it would display the options setup used to run the last test.
For this example, click Close to return to the Main Screen without saving this setup.

18. To delete this example, select the Select Group option from the Group Menu.

19. The SELECT screen appears.
Select DEFAULT. Click Continue. You are returned to the Main Screen.

20. Select the Delete Group from the Group Menu.

A SELECT screen appears. Select LOT42. Click Delete. You are returned to the Main Screen.
(The group in use cannot be deleted. In order to delete the LOT42 group, the group selected was changed to DEFAULT before clicking "Delete Group". The "Default") Locate the \capwin\data\lot42 folder. Delete the lot42 folder.
