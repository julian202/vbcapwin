CapWin Main Screen

The CapWin Main Screen displays the Group Name of the group currently in use, the Pressure Units in use, the Lohm Table in use, the menu bar of the CapWin Software System, and sometimes the type of gas being used for testing.

Each menu item (Group, Execute, Report, Modify, Calibrate, Help) displays a submenu of further options which allow you to control the porometer, report on collected data, and modify testing parameters. The more commonly used functions, such as Automated Test and Report, have shortcut keys (the Function keys on the top of the keyboard) for faster execution. These shortcuts are listed next to the option name in the relevant menu.
