Execute Menu

The Execute Menu is used to control the various functions of the porometer.

AutoTest

The Autotest function is used to set up and run an automated test. The main blue screen shown below allows the user to set the options that directly control the test.

Shortcut to the Edit Auto Parameters option from the Modify menu from the CAPWIN Main Screen. First click on Dry Curve Testing Parameters File or Wet Curve Testing Parameters File to pick a relevant file, then click this button. For a detailed explanation of this option, please refer Modify Menu.

Shortcut to using Select Group option from the Group menu ⇒ menu from the CAPWIN Main Screen.

Starts the CAPWIN Help System.

Accesses the preferences window, which allows the user to modify different settings of the specific test being used (Microflow, Liquid Permeability, Gas Permeability, and Pressure Hold) and general test setting.

Cancels the automated test setup and returns to the CAPWIN Main Screen
