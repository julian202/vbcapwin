Bubble Point Test Quick Start Sheet

Back to Quick Start Sheet TOC
  1. Open sample chamber. Disconnect the air hose attached to the sample chamber lid. Unscrew and remove the lid. Take out the spacing insert and adapter plates (if provided). Check the o-rings for dryness, apply vacuum grease if needed. Make sure components are clean and free from wetting fluid left by previous testing.

  2. Prepare sample to be tested. Cut sample to fit in chamber, if necessary. Make sure sample completely covers o-ring. Use adapter plates, if necessary.

  3. Select Group. The current group is listed on the main screen. If this is the correct group, skip this step. To change to a different group, click on the Group menu. Choose an existing group (Select Group Ctrl+U) or make a new group (New Group Ctrl+N).

  4. Under Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.
    If the Test Type field is visible but does not indicate Bubble Point Test - Automated Mode, click on Test Type to bring up the Test Selection box.
    Select Bubble Point and click on Automatic Detection under Bubble Point Options . Check the Show results at the end of test box if desired. This option will display the Bubble Point diameter and pressure immediately following completion of test.
    Click OK to return to the Autotest screen.

  5. At the Autotest screen, enter information in the designated fields by clicking on the field: Note: Depending on which fields the supervisor has check-marked, not all fields may appear in Autotest. Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Bubble Point
    Sample ID - user designated
    Lot Number - user designated
    Operator- user designated, eg: the initials of person running test
    Fluid- the wetting fluid used to saturate the sample
    Surface Tension - surface tension of the fluid, in dynes/cm .
    eg: Porewick 16.0, Galwick 15.9, Silwick 20.1, Water 72.0.

  6. Place saturated sample in sample chamber or holder, making sure sample seals against o-ring.
    If adapter plates are used, place the screened adapter plate on the bottom of sample chamber. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in putting in and taking out plates from the chamber. Place sample on top of the bottom screened plate. Make sure the o-ring of this bottom plate seals against the sample. Place top adapter plate into chamber.
    Take spacing insert and place in chamber. Avoid covering opening of insert when placing it in chamber, as this may create pressure, which will push wetting fluid from the pores of the sample and result in inaccurate measurements. The insert should sit no lower than the height of the sample chamber. Screw on lid and hand-tighten. Connect air hose from shroud to quick-connect fitting on lid.

  7. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option has been selected, the test will not require further user input and will proceed to the actual test. Otherwise, a dialogue box will appear, instructing user to Load saturated sample, seal chamber.

  8. Click OK to start test. This will bring up the Current Test Status screen. A second dialogue box will appear, prompting user to press OK to start the test.

  9. The Bubble Point test begins. Air at a set flow rate is supplied to the sample chamber. This results in a constant rate of pressure increase. A calculated value of (Flow Rate/Pressure Change)*Time, the (F/P)*T factor, appears on screen. The (F/P)*T factor grows larger as the rate of pressurization grows smaller. Once the F/P*T factor is above the set F/P*T value, the bubble point pressure is recorded and the test ends.

  10. The Test Done dialogue box will now appear. Click OK.

  11. Remove the sample from chamber. Clean chamber and components to prepare for next test run.

  12. Test results may be viewed and analyzed using CapRep. To get to CapRep from the Capwin main menu. Go under Report and click on Execute Report F4.
