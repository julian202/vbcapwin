Integrity Test (If Applicable)

Back to Quick Start Sheet TOC  Hydrohead Test
  1. Prepare sample to be tested. Sample may need to be cut in order to fit and lie flat in the sample chamber. The sample should be cut bigger than the o-ring in order to completely cover the o-ring and ensure proper sealing. Samples smaller than the o-ring will require adapter plates.

  2. Disconnect any hose attached to the sample chamber lid. Unscrew and remove the sample chamber lid. Take out the spacing insert and adapter plates (if present). Check the o-rings for dryness, apply vacuum grease if needed. Make sure components are clean and free from wetting fluid left by previous testing.

  3. Under Group on the main Capwin menu, choose an existing group by clicking on Select Group Ctrl+U or make a new group by clicking on New Group Ctrl+N. A group contains the pressure units used and the Autotest setup for the last test ran using this group.

  4. Under Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.

  5. If the Test Type field is visible but does not show Integrety, click on Test Type to bring up the Test Selection box.Select Integrity (if applicable). Check the Show results at the end of test box, if desired. This option will display the results at the end of the test. Click OK to return to the Autotest screen.

  6. At the Autotest screen, enter information in the designated fields by clicking on the field:
    Note: Depending on which fields the supervisor has selected to be shown, not all fields may appear in Autotest.
    Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Gas Permeability
    Sample ID - user designated
    Lot Number - user designated
    Operator - user designated, eg: the initials of person running test
    Fluid - enter the name of the fluid being used
    Surface Tension - enter the surface tension of the fluid being used.


  7. Place the sample into the sample chamber, making sure sample seals against o-ring.If adapter plates are used, place the screened adapter plate in the bottom of sample chamber, screen side up. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in putting in and taking out plates from the chamber. Place the sample on top of the screened plate. Make sure the o-ring of this bottom plate seals against the sample. Place the top adapter plate into the chamber.

  8. Place the chamber insert into the chamber. The insert should sit no lower than the height of the sample chamber. Squirt a uniform layer of fluid onto the sample, making sure to completely saturate and cover any exposed sample area. Screw on the lid and hand tighten. Attach the air hose with the quick connect fitting to the sample chamber lid.

  9. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option was selected, the test will not require further user input and will proceed to the actual test. Otherwise, a dialogue box will appear, instructing the user to Load sample.

  10. Click OK. This will bring up the Current Test Status screen. A second dialogue box will appear. Click OK once again to start the test. After the test is completed, the Test Done dialogue box will appear. Click OK.

  11. Remove sample from chamber. Clean chamber and components to prepare for the next test run.

  12. Test results may be viewed and analyzed using CapRep. To access CapRep from the Capwin main menu, select Report and click on Execute Report F4.
