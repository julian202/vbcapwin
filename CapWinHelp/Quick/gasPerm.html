Gas Permeability Test for Gas Permeameter

Back to Quick Start Sheet TOC Envelope Surface Area Test
  1. Prepare sample to be tested. Sample may need to be cut in order to fit and lie flat in the sample chamber. The sample should be cut bigger than the o-ring in order to completely cover the o-ring and ensure proper sealing. Samples smaller than the o-ring will require adapter plates.

  2. Disconnect any hose attached to the sample chamber lid. Unscrew and remove the sample chamber lid. Take out the spacing insert and adapter plates (if present). Check the o-rings for dryness, apply vacuum grease if needed. Make sure components are clean and free from wetting fluid left by previous testing.

  3. Under Group on the main Capwin menu, choose an existing group by clicking on Select Group Ctrl+U or make a new group by clicking on New Group Ctrl+N. A group contains the pressure units used and the Autotest setup for the last test ran using this group.

  4. Under Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.

  5. If the Test Type field is visible but does not show Gas Permeability, click on Test Type to bring up the Test Selection box.

  6. Select Gas Permeametry. Check the Show results at the end of test box, if desired. This option will display the average Darcy permeability constant immediately following completion of test.
    Click OK to return to the Autotest screen.

  7. At the Autotest screen, enter information in the designated fields by clicking on the field:
    Note: Depending on which fields the supervisor has selected to be shown, not all fields may appear in Autotest.
    Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Gas Permeability
    Sample ID - user designated
    Lot Number - user designated
    Operator - user designated, eg: the initials of person running test
    Diameter - diameter at which the sample is being tested, which is the inner diameter of the o-ring
    Thickness - thickness of sample
    *Cylindrical Length - only required for samples tested in the radial direction


  8. If the Gas Permeability test is being performed using a gas other than air and the Gas field is visible, click on the Gas field. This will bring up the Gas Select box. Select X: Other gas and enter the name of the gas and its viscosity in centiPoise. Click Done to return to the Autotest screen.

  9. Place sample in sample chamber, making sure sample seals against o-ring.If adapter plates are used, place the screened adapter plate on the bottom of sample chamber. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in putting in and taking out plates from the chamber. Place sample on top of bottom screened plate. Make sure the o-ring of this bottom plate seals against the sample. Place top adapter plate into chamber.
    Take spacing insert and place into chamber. The insert should sit no lower than the height of the sample chamber. Screw on lid and hand-tighten. Connect hose to quick-connect fitting on lid.

  10. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option was selected, the test will not require further user input and will proceed to the actual test. Otherwise, a dialogue box will appear, instructing user to Load sample, seal chamber.

  11. Click OK to start test. This will bring up the Current Test Status screen. A second dialogue box will appear, prompting user to press OK to start the test.

  12. The Gas Permeability test begins. The test will increase the pressure and measure the flow through and pressure drop across the dry sample. The graph is plotted on the screen as the test progresses. The test ends upon reaching the set maximum pressure or flow rate.

  13. At the completion of the test, the Test Done dialogue box will appear. ClickOK.

  14. Remove sample from chamber. Clean chamber and components to prepare for the next test run.

  15. Test results may be viewed and analyzed using CapRep. To access CapRep from the Capwin main menu, select Report and click on Execute Report F4.
