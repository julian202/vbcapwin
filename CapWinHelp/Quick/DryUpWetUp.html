Dry Up/Wet Up Test for Top-Down Porometers

Back to Quick Start Sheet TOC Bubble Point Test
  1. Prepare the sample to be tested. The sample may need to be cut in order to fit and lie flat in the sample chamber. The sample should be cut bigger than the o-ring in order to completely cover the o-ring and ensure proper sealing. Samples smaller than the o-ring will require adapter plates.

  2. Disconnect the air hose attached to the sample chamber lid. Unscrew and remove the lid. Take out the spacing insert and adapter plates (if provided). Check the o-rings for dryness, apply vacuum grease if needed. Make sure all components are clean and free from wetting fluid left by previous testing.

  3. Under Group on the main Capwin menu, choose an existing group by clicking on Select Group Ctrl+U or make a new group by clicking on New Group Ctrl+N.
    A group contains the pressure units used and the Autotest setup for the last test ran using this group.

  4. Under Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.

  5. If the Test Type field is visible but does not show Capillary Flow Porometry -Dry up, Wet up, click on Test Type to bring up the Test Selection box.
    Select Capillary Flow Porometry and click on Dry Up/Wet Up under Capillary Flow Porometry Options . Check the Show results at the end of test box if desired. This option will display the Bubble Point and Mean Flow Pore pressures and diameters immediately following completion of the test.

  6. At the Autotest screen, enter information in the designated fields by clicking on the field:
    Note: Depending on which fields the supervisor has selected to be shown, not all fields may appear in Autotest.
    Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Capillary Flow Porometry, or Dry Up/Wet Up
    Sample ID - user designated
    Lot Number - user designated
    Operator - user designated, eg: the initials of person running test
    Fluid - the wetting fluid used to saturate the sample during the Wet Up portion of the test.
    Surface Tension - surface tension of the fluid, in dynes/cm.
    eg: Porewick 19., Galwick 15.6, Silwick 20.1, Water 72.0.
    Diameter - diameter at which the sample is being tested, which is the inner diameter of the o-ring.
    Thickness - thickness of sample
    *Cylindrical Length - only required for samples tested in the radial direction

  7. Place the sample into the sample chamber, making sure the sample seals against the o-ring.
    If adapter plates are used, place the screened adapter plate on the bottom of sample chamber. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in putting in and taking out plates from the chamber. Place the sample on top of the screened plate. Make sure the o-ring of the bottom plate seals against the sample. Place the top adapter plate into chamber.


  8. Place the chamber insert in the chamber. The insert should sit no lower than the height of the sample chamber. Screw on the lid and hand tighten. Connect the air hose to the quick-connect fitting on the lid.

  9. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option was selected, the test will not require further user input and will proceed to the actual test. Otherwise, a dialogue box will appear, instructing the user to Load sample, seal chamber.
    Click OK to start the test. This will bring up the Current Test Status screen. A second dialogue box will appear, prompting user to press OK to start the test.

  10. The dry portion of the Dry Up/Wet Up test begins. The test will increase the pressure and measure the flow through and pressure drop across the dry sample. The graph is plotted on the screen as test progresses. At the end of the dry curve, test will pause and wait for the user to saturate the sample.

  11. A dialogue box will appear, prompting the user to Load saturated sample, seal chamber.
    Saturate sample with wetting fluid. It is preferable to saturate the sample by unscrewing the lid and squirting wetting fluid onto the sample through the opening of the insert. This lessens the chance of the sample moving during testing. If the sample needs to be taken out of the chamber to saturate, check that the sample still covers the o-ring when it is placed back into the chamber. Avoid covering the opening of the insert when replacing it back into the chamber , as this may create pressure, which may push wetting fluid from the pores of the sample and result in inaccurate measurements.

  12. Press OK after loading the sample. A second dialogue box will appear. Clicking OK the second time will start the wet portion of the test.

  13. At the completion of the wet curve, the Test Done dialogue box will appear. Click OK.

  14. Remove the sample from the chamber. Clean the chamber and all components to prepare for the next test run.

  15. Test results may be viewed and analyzed using CapRep. To get to CapRep from the Capwin main menu. Go under Report and click on Execute Report F4.
