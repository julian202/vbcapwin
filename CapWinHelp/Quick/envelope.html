Envelope Surface Area for Top-Down Porometers

Back to Quick Start Sheet TOC  Dry Up Wet Up Test
  1. Prepare the sample to be tested. In envelope surface area testing, you may test either fabrics or powdered samples. To test fabric samples use the porometers sample chamber. To test powders you must use the envelope surface area sample chamber. Refer to Section D, Test Procedures, for further information on sample installation.

  2. Decide which sample chamber you will use. Select either the provided chamber or the envelope surface area powder chamber. Load the sample as described in the reference above.

  3. Under Group on the main Capwin menu, choose an existing group by clicking on Select Group Ctrl+U or make a new group by clicking on New Group Ctrl+N. A group contains the pressure units used and the Autotest setup for the last test ran using this group.

  4. Under Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.

  5. If the Test Type field is visible but does not show Envelope Surface Area Analysis, click on Test Type to bring up the Test Selection box. Select Envelope Surface Area. Check the Show results at the end of test box if desired. This option will display the average surface area, fiber diameter and spherical particle diameter immediately following completion of test.
    Click OK to return to the Autotest screen.

  6. At the Autotest screen, enter information in the designated fields by clicking on the field:
    Note: Depending on which fields the supervisor has selected to be shown, not all fields may appear in Autotest.
    Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Envelope Surface Area
    Sample ID - user designated
    Lot Number - user designated
    Operator - user designated, eg: the initials of person running test
    Gas - the gas used for this test
    Diameter - diameter at which the sample is being tested, which is the inner diameter of the o-ring
    Thickness - thickness of sample
    Abs. Density - absolute density of the sample, if known, otherwise enter 1
    Mass - mass of the cut circular sample
    Sample diameter for measured mass - the diameter of the cut circular sample (may be larger than the test diameter)
    Steps 7 and 8 for normal sample chamber only

  7. Place sample in sample chamber, making sure sample seals against o-ring.If adapter plates are used, place the screened adapter plate on the bottom of sample chamber. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in putting in and taking out plates from the chamber. Place sample on top of bottom screened plate. Make sure the o-ring of this bottom plate seals against the sample. Place top adapter plate into chamber.

  8. Place chamber insert into the chamber. The insert should sit no lower than the height of the sample chamber. Screw on the lid and hand-tighten. Connect the air hose to the quick-connect fitting on the lid.

  9. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option was selected, the test will not require further user input and will proceed to the actual test. Otherwise, a dialogue box will appear, instructing the user to Load sample, seal chamber.

  10. 10. Click OK to start the test. This will bring up the Current Test Status screen. A second dialogue box will appear, prompting the user to press OK to start the test.

  11. The Envelope Surface Area test begins. The test will increase the pressure and measure the flow through and pressure drop across the dry sample. The graph is plotted on the screen as the test progresses. The test ends upon reaching the set maximum pressure or flow rate.

  12. At the completion of the test, the Test Done dialogue box will appear. Click OK.

  13. Test results may be viewed and analyzed using CapRep. To access CapRep from the Capwin main menu, select Report and click on Execute Report F4.
