Leak Test For Capillary Flow Porometers

Back to Quick Start Sheet TOC  Integrity Test

To ensure that the instrument system is leak tight, perform a leak test before proceeding to any automated test procedures. Leak tests can also be used as a way of diagnosing instrument problems. The user specifies the maximum pressure of the leak test, the step pressure, and the wait time at each step.

Chamber Preparation
  1. Disconnect the air hose attached to the sample chamber lid. Unscrew and remove the lid. Take out the chamber insert, any adapter plates and samples that may be left in the sample chamber.

  2. Check the o-rings for dryness, apply vacuum grease if needed. Make sure the components are clean and free from wetting fluid left by previous testing.

Leak Test Settings
  1. Under Execute on the main Capwin menu, select Leak Test (or press Ctrl + L). A series of dialog boxes will appear.

  2. 4. The LeakTest Setup dialog box opens, prompting the user to Place Non-porous Insert in Chamber and Seal.
    Take the non-porous metal or plastic disk and place it in the sample chamber. In order to ensure proper sealing between the non-porous disk and the top and bottom o-rings, make sure that the area of the disk covering the o-ring is smooth and free from deep scratches. The disk should completely cover the o-rings.
    Place the chamber insert into the chamber. The insert should sit no lower than the height of the sample chamber. Screw on the lid and hand-tighten. Connect the air hose to the quick-connect fitting on the lid.
    Click OK to continue.

  3. A dialog box appears, prompting the user to Enter Maximum Pressure (differential).
    Enter the maximum pressure of the leak test. Since this value is the differential pressure, the instrument will add on the current atmospheric pressure reading. In determining what value to use, set the maximum pressure to a pressure greater than the maximum pressure you anticipate to use for the automated testing, but no more than the maximum pressure of your inlet pressure supply.
    Click OK to continue.

  4. A dialog box appears, prompting the user to Enter Step Pressure.
    Enter the step pressure for the leak test. The step pressure is the amount of pressure that will be increased at each step interval. This can be set to any value up to or including the maximum pressure of the leak test. When the step pressure is set equal to the maximum pressure of the leak test, the instrument will attain the maximum pressure of the leak test in 1 step and hold for the duration of the test. Another common practice is to use a value which will yield 10 pressure steps.
    Click OK to continue.

  5. A dialog box appears, prompting the user to Enter Wait Time (min).
    If the step pressure is set to the maximum pressure of the leak test, then the wait time is the test duration. Otherwise the pressure step will continue to increase at the time specified by the user in this input.
    Click OK to continue.

Run LeakTest
  1. The LeakTest Setup dialog box appears, indicating the summary of the leak test procedure: The program will acquire readings from X to Y Pressure Units (Gauge), in X Pressure Unit increments, waiting Z minutes between each point.
    Click OK.
    The instrument proceeds to increase the pressure to the first step and holds for the duration of the wait time.

  2. At the end of the leak test, the instrument will vent off the pressure. The LeakTest dialog box appears, stating the instrument passed [or failed] the leak test over the range of X to Y Pressure Units (differential). Do you wish to see the collected data?
    Click Yes if you want to view the data.

  3. 10. The Done dialog box appears, stating LeakTest Finished. Remove Non-porous Insert. Click OK.

View LeakTest results
  1. If Yes was clicked, Notepad opens to show the LeakTest results. The results are stored in the file Leaktest.asc, under the CapWin directory, and are viewed in Notepad. Leak test data is reported as a percentage of leak per hour. PI is the pressure at the beginning of the wait time at each step, and PF is the pressure taken at the end of each wait time. Normally, a leak rate within 7% per hour is acceptable.

  2. Save the file under a different name if you do not want the data to be overwritten the next time the leak test is run. Exit Notepad when you are finished viewing the data.
