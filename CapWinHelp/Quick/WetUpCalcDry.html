Wet Up/Calculated Dry Test

Back to Quick Start Sheet TOC  Pressure Hold Test

Wet Up/Calculated Dry is different from the other Capillary Flow Porometry Options in that Wet Up/Calculated Dry does not have data points for the dry curve. Instead, it calculates and then plots a dry curve using points from the wet curve. Use of this test is advantageous in cases when only a wet curve is needed. Wet and dry curves from different files may be combined into one file in Data Editor.

The CapWin software offers three options for calculated dry curves: Linear, Darcy, and Lohm.

Linear plots a line connecting the last point of the wet curve with the origin. Use this for samples which have a dry curve that is more or less linear.

Darcy uses the Darcy equation and points from the wet curve to plot a parabolically shaped line, symmetric along the y axis. Use this for samples which show a similarly shaped dry curve. Samples suitable for use with a Darcy dry curve typically go to high pressures and but have low flow.

Square Root uses an equation which is based on the square root function.

  1. Prepare sample to be tested. Sample may need to be cut in order to fit and lie flat in the sample chamber. The sample should be cut bigger than the o-ring in order to completely cover the o-ring and ensure proper sealing. Samples smaller than the o-ring will require adapter plates.

  2. Disconnect any hose attached to the sample chamber lid. Unscrew and remove the sample chamber lid. Take out the spacing insert and adapter plates (if present). Check the o-rings for dryness, apply vacuum grease if needed. Make sure components are clean and free from wetting fluid left by previous testing.

  3. Under Group on the main Capwin menu, choose an existing group by clicking on Select Group Ctrl+U or make a new group by clicking on New Group Ctrl+N. A group contains the pressure units used and the Autotest setup for the last test ran using this group.

  4. Under Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.

  5. If the Test Type field is visible but does not show Capillary Flow Porometry - Wet up, Dry Up Linear click on Test Type to bring up the Test Selection box.
    Select Capillary Flow Porometry and click on Wet Up/Calc. Dry under Capillary Flow Porometry Options. Choose one of the three methods (refer to top of page for more info). Check the Show results at the end of test box if desired. This option will display the Bubble Point and Mean Flow Pore pressures and diameters immediately following completion of test.
    Click OK to return to the Autotest screen.

  6. At the Autotest screen, enter information in the designated fields by clicking on the field:
    Note: Depending on which fields the supervisor has selected to be shown, not all fields may appear in Autotest.
    Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Gas Permeability
    Sample ID - user designated
    Lot Number - user designated
    Operator - user designated, eg: the initials of person running test
    Fluid - the wetting fluid used to saturate the sample during the Wet Up portion of the test.
    Surface Tension - surface tension of the fluid, in dynes/cm.
    eg: Porewick ~ 16.1, Galwick ~ 15.6, Silwick ~ 20.1, Water 72.0.
    Diameter - diameter at which the sample is being tested, which is the inner diameter of the o-ring.
    Thickness - Thickness of sample
    Cylindrical Length -only required for samples tested in the radial direction


  7. Place saturated sample in sample chamber, making sure sample seals against o-ring.
    If adapter plates are used, place the screened adapter plate on the bottom of sample chamber. The o-ring on the bottom plate should face upwards. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in placement of adapter plates. Put in the saturated sample. Make sure sample fully covers the o-ring of the bottom plate in order to make a good seal. Place top adapter plate into chamber with o-ring facing downward. Sample should now be completely sealed against the top and bottom o-rings of the adapter plates.

  8. Take spacing insert and place into chamber. Avoid covering opening of the insert when placing it in chamber, as this may create pressure which will push wetting fluid from the pores of the sample and result in inaccurate measurements. The bottom o-ring of the spacing insert should be in contact with the sample (or the adapter plate, if used). The insert should sit no lower than the height of the sample chamber. Screw on lid and hand tighten. Connect the air hose from shroud to quick-connect fitting on lid.

  9. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option was selected, the test will not require further user input and will proceed to the actual test. Otherwise, a dialogue box will appear, instructing user to Load sample, seal chamber.

  10. Click OK to start test. This will bring up the Current Test Status screen. A second dialogue box will appear, prompting user to press OK to start the test.

  11. The Wet Up portion of the Wet Up/Calc. Dry test begins. The test will increase the pressure and measure the flow through and pressure drop across the saturated sample. The graph is plotted on the screen as test progresses.

  12. At the completion of the wet curve, the calculated dry curve is plotted. Then the Test Done dialogue box will appear. Click OK.

  13. Remove sample from chamber. Clean chamber and components to prepare for next test run.

  14. Test results may be viewed and analyzed using CapRep. To access CapRep from the Capwin main menu, select Report and click on Execute Report F4.
