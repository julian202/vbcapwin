Pressure Hold Test

  1. Prepare sample to be tested. Sample may need to be cut in order to fit and lie flat in the sample chamber. The sample should be cut bigger than the o-ring in order to completely cover the o-ring and ensure proper sealing. Samples smaller than the o-ring will require adapter plates.

  2. Prepare sample to be tested. Sample may need to be cut in order to fit and lie flat in the sample chamber. The sample should be cut bigger than the o-ring in order to completely cover the o-ring and ensure proper sealing. Samples smaller than the o-ring will require adapter plates.

  3. Under Group on the main Capwin menu, choose an existing group by clicking on Select Group Ctrl+U or make a new group by clicking on New Group Ctrl+N. A group contains the pressure units used and the Autotest setup for the last test ran using this group.

  4. Under Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.
    If the Test Type field is visible and Pressure Hold Test is not shown, then click on the Test Type. This will bring up the Test Selection box. Select Pressure Hold.
    Click OK to return to the Autotest screen.

  5. At the Autotest screen, enter information in the designated fields by clicking on the field:
    Note: Depending on which fields the supervisor has selected to be shown, not all fields may appear in Autotest.
    Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Gas Permeability
    Sample ID - user designated
    Lot Number - user designated
    Operator - user designated, eg: the initials of person running test
    Gas - name of the gas used for test.
    Hold Parameters- Starting pressure differential


  6. At the Preferences screen, click the Pressure Hold tab and you can change the Time and Delay parameters, as well as adding a Pass/Fail option.

  7. Place the sample into the sample chamber, making sure sample seals against o-ring.
    If adapter plates are used, place the screened adapter plate in the bottom of the sample chamber. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in putting in and taking out plates from the chamber. Place the sample on top of the screened plate. Make sure the o-ring of this screened plate seals against the sample. Place the top adapter plate into the chamber.

  8. Take spacing insert and place into chamber. The insert should sit no lower than the height of the sample chamber. Screw on lid and hand tighten. Attach the air hose with quick connect fitting to the sample chamber lid.

  9. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option was selected, the test will not require further user input and will proceed to the actual test. A dialogue box will appear, instructing user to Load sample, seal chamber.

  10. Click OK. This will bring up the Current Test Status screen and will start the test. The pressure is increased until the starting pressure, as designated on the Autotest screen, is reached. Data is taken at intervals.

  11. Upon reaching the set time limit for the test, the Test Done dialogue box will appear. Click OK.

  12. Remove sample from chamber. Clean chamber and components to prepare for next test run.

  13. Test results may be viewed and analyzed using CapRep. To access CapRep from the Capwin main menu, select Report and click on Execute Report F4.
