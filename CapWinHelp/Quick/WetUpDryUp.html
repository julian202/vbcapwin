Wet Up/ Dry Up Test

Back to Quick Start Sheet TOC  Pressure Hold Test
  1. Prepare sample to be tested. Sample may need to be cut in order to fit and lie flat in the sample chamber. The sample should be cut bigger than the o-ring in order to completely cover the o-ring and ensure proper sealing. Samples smaller than the o-ring will require adapter plates.

  2. Disconnect any hose attached to the sample chamber lid. Unscrew and remove the sample chamber lid. Take out the spacing insert and adapter plates (if present). Check the o-rings for dryness, apply vacuum grease if needed. Make sure components are clean and free from wetting fluid left by previous testing.

  3. Under Group on the main Capwin menu, choose an existing group by clicking on Select Group Ctrl+U or make a new group by clicking on New Group Ctrl+N. A group contains the pressure units used and the Autotest setup for the last test ran using this group.

  4. Under Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.

  5. If the Test Type field is visible but does not show Capillary Flow Porometry -Wet up, Dry up, click on Test Type to bring up the Test Selection box.
    Select Capillary Flow Porometry and click on Wet Up/Dry Up under Capillary Flow Porometry Options.
    Check No Wait at Dry for test to continue on to the Dry Up portion of test upon reaching the set maximum pressure or flow. Otherwise, test will wait for user to press OK before starting on the Dry Up portion of test.
    Check the Show results at the end of test box, if desired. This option will display the Bubble Point and Mean Flow Pore pressures and diameters immediately following completion of test.
    Click OK to return to the Autotest screen.

  6. At the Autotest screen, enter information in the designated fields by clicking on the field:
    Note: Depending on which fields the supervisor has selected to be shown, not all fields may appear in Autotest.
    Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Capillary Flow Porometry, or Wet Up/Dry Down
    Sample ID - user designated
    Lot Number - user designated
    Operator - user designated, eg: the initials of person running test
    Fluid - the wetting fluid used to saturate the sample during the Wet Up portion of the test.
    Surface Tension - surface tension of the fluid, in dynes/cm.
    eg: Porewick ~ 16.1, Galwick ~ 15.6, Silwick ~ 20.1, Water 72.0.
    Diameter - diameter at which the sample is being tested, which is the inner diameter of the o-ring.
    Thickness - Thickness of sample
    Cylindrical Length -only required for samples tested in the radial direction


  7. Place saturated sample in sample chamber, making sure sample seals against o-ring.
    If adapter plates are used, place the screened adapter plate on the bottom of sample chamber. The o-ring on the bottom plate should face upwards. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in placement. Put in the saturated sample. Make sure sample fully covers the o-ring of the bottom plate in order to make a good seal. Place top adapter plate into chamber with o-ring facing downward. The sample should now be completely sealed against the top and bottom o-rings of the adapter plates.

  8. Take spacing insert and place into chamber. Avoid covering opening of insert when placing it in chamber, as this may create pressure, which will push wetting fluid from the pores of the sample and result in inaccurate measurements. The bottom o-ring of the spacing insert should be in contact with the sample (or the adapter plate, if used). The insert should sit no lower than the height of the sample chamber. Screw on lid and hand tighten. Connect air hose from shroud to quick-connect fitting on lid.

  9. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option was selected, test will not require further user input and will proceed to the actual test. Otherwise, a dialogue box will appear, instructing user to Load saturated sample, seal chamber.

  10. Click OK to start test. This will bring up the Current Test Status screen. A second dialogue box will appear, prompting user to press OK to start the test.

  11. The Wet Up portion of the Wet Up/Dry Up test begins. The test will increase the pressure and measure the flow through and pressure drop across the saturated sample. The graph is plotted on the screen as test progresses. If No Wait at Dry was selected, test will "turn" and will go on to the Dry Up portion of the test upon reaching the set maximum pressure or flow. Otherwise, a prompt to Load sample, seal chamber will appear when test reaches the end of the Wet Up portion of test.

  12. Test may proceed using the sample already in the chamber or with a new dry sample. This is left up to supervisor discretion.

  13. Click OK to proceed to Dry Up portion of test.

  14. At the completion of the dry curve, the Test Done dialogue box will appear. Click OK.

  15. Remove sample from chamber. Clean chamber and components to prepare for next test run.

  16. Test results may be viewed and analyzed using CapRep. To access CapRep from the Capwin main menu, select Report and click on Execute Report F4.
