Gas Permeability Test for Gas Permeameter

Back to Quick Start Sheet TOC Envelope Surface Area Test
  1. Prepare the sample to be tested. Samples to be tested for hydrohead should be new, dry samples, which have never come into contact with water. If the sample needs to be cut in order to fit and lie flat in the sample chamber, cut the sample to a size bigger than the o-ring in order to completely cover the o-ring and ensure proper sealing. Samples smaller than the o-ring will require adapter plates.

  2. Disconnect the air hose attached to the sample chamber lid. Unscrew and remove the lid. Take out the chamber insert and adapter plates (if provided). Check the o-rings for dryness, apply vacuum grease if needed. Make sure the components are clean and free from wetting fluid left by previous testing.

  3. Under Group on the main Capwin menu, choose an existing group by clicking on Select Group Ctrl+U or make a new group by clicking on New Group Ctrl+N. A group contains the pressure units used and the Autotest setup for the last test ran using this group.

  4. Under the Execute on the main Capwin menu, select Autotest F2. The Autotest settings screen will open.

  5. If the Test Type field is visible but does not show Gas Permeability, click on Test Type to bring up the Test Selection box.

  6. At the Autotest screen, enter information in the designated fields by clicking on the field:
    Note: Depending on which fields the supervisor has selected to be shown, not all fields may appear in Autotest.
    Output File Name - user designated
    End User - user designated
    Test Reference - user designated, eg: Gas Permeability
    Sample ID - user designated
    Lot Number - user designated
    Operator - user designated, eg: the initials of person running test
    Fluid - Water
    Surface Tension - 72 the surface tension of water


  7. Place the sample into the sample chamber, making sure the sample seals against the o-ring. If adapter plates are used, place the screened adapter plate on the bottom of sample chamber. Align the plate on the three chamber pins. Yellow chip pullers are provided to aid in putting in and taking out plates from the chamber. Place the sample on top of the screened plate. Make sure the o-ring of the bottom plate seals against the sample. Place the top adapter plate into the chamber.

  8. Place the chamber insert into the chamber. The insert should sit no lower than the height of the sample chamber. Squirt a uniform layer of water onto the sample. Be sure to completely cover any exposed area of the sample. Screw on the lid and hand-tighten. Attach the air hose with the quick connect fitting to the sample chamber lid.

  9. Click on the Start Test button ALT+S. The Autotest sequence will now begin. If the Hide Sample Load Prompts option was selected, the test will not require further user input and will proceed to the actual test. Otherwise, a dialogue box will appear, instructing the user to Load sample with fluid on top, seal chamber.

  10. Click OK. This will bring up the Current Test Status screen. A second dialogue box will appear. Clicking OK will begin the test.

  11. Upon finding the hydrohead pressure, the Test Done dialogue box will appear. Click OK.

  12. Remove the sample from the chamber. Clean the chamber and all components to prepare for the next test run.

  13. Test results may be viewed and analyzed using CapRep. To access CapRep from the Capwin main menu, select Report and click on Execute Report F4.
